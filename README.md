# X Post Maker

یک ابزار ساده برای ساخت کارت‌های پست شبیه به شبکه اجتماعی X (توییتر سابق) با قابلیت شخصی‌سازی کامل.

# ویژگی‌ها

- آپلود آواتار : با کلیک روی دایره آواتار، عکس دلخواه خود را آپلود کنید
- ویرایش نام و یوزرنیم : نام نمایشی و یوزرنیم را به دلخواه تغییر دهید
- متن دلخواه : هر متنی که دوست دارید بنویسید
- تاریخ و ساعت بروز : زمان و تاریخ به‌صورت خودکار از سیستم شما گرفته می‌شود
- تعداد بازدید قابل ویرایش : تعداد بازدید را خودتان تنظیم کنید
- دانلود به‌صورت تصویر : با یک کلیک، کارت را به‌صورت PNG با کیفیت بالا دانلود کنید
